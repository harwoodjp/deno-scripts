### To run

`deno run [args] [script]`
* args
	* `--allow-run`
* script
	* File
	* URL
	* `.ts`
	* `.js`

### Scripts

`runCommand.js`
* Run input command as Deno subprocess

`scratchts`
* Create scratch files for notes, code experiments, etc.
* `deno run --allow-read --allow-write --allow-run scratch.ts py`
